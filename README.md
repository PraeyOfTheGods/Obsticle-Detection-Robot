# Obsticle-Detection-Robot
Just my College project
For FRA (Fundamentals of Robotics and Applications) 
The project contains Program, project image
Components are:
Arduino Nano (ATmega328P)
HC-SR04 Ultrasonic Sensor
SG90 Servo Motor
DC Geared Motors (2 units)
L298N Dual H-Bridge Motor Driver Module
6V Battery Pack (4× AA batteries)
USB / 5V Supply (for Arduino Nano)
Robot Chassis
Wheels (2 units)
Jumper Wires (Male–Male, Male–Female)
Screws, Nuts, and Spacers
Arduino IDE
Servo Library
